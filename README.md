# CareTaker チームプロジェクト

概要

    「仕事が忙しい30代の独身男性」をペルソナにしたアプリの開発
    　・仕事が忙しすぎ、髪・髭が伸びている、土日は寝て過ごすゴミの捨て忘れに気が付かない
     
    「リマインダー」の応用
    　・項目の追加や通知


ゴール
    
    ・swiftアジャイル開発ができる

    ・チーム内のコミュニケーションが上手くできている

    ・MVPを最優先

    ・テストを書く

    ・フルスタックはマスト

    ・ユーザーのカスタム機能も追加

    ・プッシュ通知

    ・Backendとの連携

    ・xcodeでペアプロ

    ・CI/CD

    ・UI/UX


機能

    ・リマインダー
        1.　各項目の実施日付を更新可
        2.　各項目の実施周期の選択可
        3.　設定した周期に基づいて色が変化する（白→赤）
        
        
    ・データベース
        1.　Firebaseを活用
        2.　各項目の情報の更新を行う
